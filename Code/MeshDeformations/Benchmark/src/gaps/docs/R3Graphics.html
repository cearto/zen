R3Graphics Package

Description:

The R3Graphics package supports classes for basic computer graphics: 3D viewing, camera control, surface material properties, lighting, and scene graphs.

List of Classes:

  • Camera and viewport control:
    R2Viewport - 2D screen viewport.
    R3Camera - 3D camera (view frustum).
    R3Viewer - Mapping from 3D camera to 2D viewport.
  • Surface material properties:
    R3Brdf - 3D bidirection reflectance distribution function.
    R2Texture - 2D surface texture.
    R3Material - 3D material properties (brdf, texture, and texture transformation).
  • Lights:
    R3Light - Light abstract class.
    R3DirectionalLight - Directional light.
    R3PointLight - Point light.
    R3SpotLight - Spot light.
  • Scene graphs:
    R3Scene - scene graph.
    R3Node - scene graph node.
