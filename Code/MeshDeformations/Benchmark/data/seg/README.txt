Files for human and computer generated segmentations are provided here.

HUMAN-GENERATED SEGMENTATIONS
=============================
./Benchmark
380 directories each containing multiple human segmentations for that model

Vis.Comput. 22, 3, 181–193.

./NormCuts

./RandWalks

In Symposium on Solid and Physical Modeling, 183–191.

./ShapeDiam

Consistent mesh partitioning and skeletonisation using the shape diameter function.
