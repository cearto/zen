
DIRECTORIES
===========

./apps
Source code of programs used in segmentation evaluation and anslysis

./freeglut
Library for GLUT

./gaps
Library on which our application programs are based


COMPILATION
===========

Linux: 
Type "make" in the current directory, the executables will be in
"../exe/"


Windows: 
Open the Visual Studio 2005 solution file "src.sln" in the current directory
and recompile everything. All executables will be in "../exe/"
 
Pre-compiled win32 library and application programs are already included in
"./gaps/lib/win32/" and "../exe/"
